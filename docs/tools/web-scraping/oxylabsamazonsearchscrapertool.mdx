# `OxylabsAmazonSearchScraperTool`


## Installation

```shell
pip install 'crewai[tools]' oxylabs
```

Get the credentials by creating an Oxylabs Account [here](https://oxylabs.io).

## Example

```python
from crewai_tools import OxylabsAmazonSearchScraperTool

# make sure OXYLABS_USERNAME and OXYLABS_PASSWORD variables are set
tool = OxylabsAmazonSearchScraperTool()

result = tool.run(query="headsets")

print(result)
```

## Parameters

- `query` - Amazon search term.
- `domain` - Domain localization for Bestbuy. The full list of available domains can be found [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/localization/domain-locale-results-language#domain).
- `start_page` - starting page number.
- `pages` - number of pages to retrieve.
- `geo_location` - the _Deliver to_ location. See our guide to using this parameter [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/localization/e-commerce-localization).
- `user_agent_type` - device type and browser. The full list can be found [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/http-context-and-job-management/user-agent-type).
- `render` - enables JavaScript rendering when set to `html`. Learn more [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/js-rendering-and-browser-control/javascript-rendering).
- `callback_url` - URL to your callback endpoint. Learn more [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/integration-methods/push-pull#callback).
- `context` - Additional advanced settings and controls for specialized requirements. Learn more [here](https://developers.oxylabs.io/scraping-solutions/web-scraper-api/targets/amazon/search#other).
- `parse` - returns parsed data when set to true. Explore output [data dictionary](https://developers.oxylabs.io/scraper-apis/web-scraper-api/targets/amazon/search#output-data-dictionary).
- `parsing_instructions` - define your own parsing and data transformation logic that will be executed on an HTML scraping result. Learn more [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/result-processing-and-storage/custom-parser/parsing-instruction-examples).

## Advanced example

Check out the Oxylabs [documentation](https://developers.oxylabs.io/scraper-apis/web-scraper-api/targets/amazon/search) to get the full list of parameters.

```python
from crewai_tools import OxylabsAmazonSearchScraperTool

# make sure OXYLABS_USERNAME and OXYLABS_PASSWORD variables are set
tool = OxylabsAmazonSearchScraperTool(
    config={
        "domain": 'nl',
        "start_page": 2,
        "pages": 2,
        "parse": True,
        "context": [
            {'key': 'category_id', 'value': 16391693031}
        ],
    }
)

result = tool.run(query='nirvana tshirt')

print(result)
```
