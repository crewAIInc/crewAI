# `OxylabsAmazonProductScraperTool`


## Installation

```shell
pip install 'crewai[tools]' oxylabs
```

Get the credentials by creating an Oxylabs Account [here](https://oxylabs.io).

## Example

```python
from crewai_tools import OxylabsAmazonProductScraperTool

tool = OxylabsAmazonProductScraperTool(
    username="OXYLABS_USERNAME",
    password="OXYLABS_PASSWORD",
)
result = tool.run(query="AAAAABBBBCC")

print(result.results[0].content)
```

## Parameters

- `query` - 10-symbol ASIN code.
- `domain` - domain localization for Amazon. The full list of available domains can be found [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/localization/domain-locale-results-language#domain).
- `geo_location` - the _Deliver to_ location. See our guide to using this parameter [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/localization/e-commerce-localization).
- `user_agent_type` - device type and browser. The full list can be found [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/http-context-and-job-management/user-agent-type).
- `render` - enables JavaScript rendering when set to `html`. Learn more [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/features/js-rendering-and-browser-control/javascript-rendering).
- `callback_url` - URL to your callback endpoint. Learn more [here](https://developers.oxylabs.io/scraper-apis/web-scraper-api/integration-methods/push-pull#callback).
- `parse` - returns parsed data when set to true. Explore output [data dictionary](https://developers.oxylabs.io/scraper-apis/web-scraper-api/targets/amazon/product#output-data-dictionary).

## Advanced example

Check out the Oxylabs [documentation](https://developers.oxylabs.io/scraper-apis/web-scraper-api/targets/amazon/product) to get the full list of parameters.

```python
from crewai_tools import OxylabsAmazonProductScraperTool

tool = OxylabsAmazonProductScraperTool(
    username="OXYLABS_USERNAME",
    password="OXYLABS_PASSWORD",
)
result = tool.run(
    query="AAAAABBBBCC",
    domain="nl",
    parse=True,
    context=[
        {
            "key": "autoselect_variant",
            "value": True
        }
    ]
)

print(result.results[0].content)
```
