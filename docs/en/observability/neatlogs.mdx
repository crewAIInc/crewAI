## Quick Setup

<Steps>
  <Step title="Sign up and get your API key">
    Go to [neatlogs.com](https://neatlogs.com/?utm_source=crewAI-docs), create a project, and copy the API key from your dashboard.
  </Step>
  <Step title="Install the SDK">
    ```bash
    pip install neatlogs
    ```
  </Step>
  <Step title="Add this before your Crew runs">
    ```python
    import os
    import neatlogs

    neatlogs.init(
        api_key=os.environ["NEATLOGS_API_KEY"],
        endpoint=os.environ["NEATLOGS_ENDPOINT"],
        workflow_name="my-agent",
        instrumentations=["openai"],
    )
    ```

    That's it. Every agent run is now captured automatically with no changes to the rest of your code.
  </Step>
</Steps>
