## 数据质量评分
**整体评分：6.5/10**

**评分理由：**
数据框架基本完整，包含了M5、M15周期的K线及常用技术指标。然而，数据存在明显的结构性缺陷：M5和M15周期的K线数据量严重不足，无法进行有效的趋势和形态分析；成交量数据仅以原始值和相对比率呈现，缺乏加权和累积分析；技术指标参数单一，缺少多时间框架的共振分析；关键的价格结构（支撑/阻力）信息缺失，且部分指标信号存在矛盾，降低了数据对决策的直接支持价值。

## 数据问题清单

| 问题描述 | 严重程度 | 影响范围 |
| :--- | :--- | :--- |
| **K线数据量严重不足**：`m5_candles`仅提供12根（1小时），`m15_candles`仅提供20根（5小时）。对于EMA200等长期指标的计算和形态识别而言，数据量远远不够。 | 高 | 趋势判断、指标可靠性、形态识别 |
| **成交量分析维度单一**：仅有单根K线的成交量(`v`)和相对于近期均量的比率(`vol_ratio`)，缺少成交量加权平均价(VWAP)、累积成交量指标(OBV)及成交量分布分析。 | 中 | 价格有效性验证、突破确认、市场参与度判断 |
| **技术指标参数固化且不完整**：EMA仅提供20/50/200三个周期，缺少对短期动能（如EMA9）和中期趋势（如EMA100）的捕捉。动量指标中，MACD仅提供状态描述，无具体数值；ADX有数值但无方向线(±DI)。 | 中 | 多周期趋势分析、入场/出场点位细化 |
| **关键价格位缺失**：数据中仅包含周线级别的单一关键支撑位(`w1_key_support`)，缺少当前交易区间内（如H4、D1）的动态支撑/阻力、斐波那契回撤位、近期高低点等关键价格信息。 | 高 | 风险管理、目标位设定、订单布局 |
| **指标间存在信号矛盾**：`m15_1000_bars`的`ema_cluster`显示EMA200(2094.15)低于当前价格(2094.4)，但`alignment`状态为“缠绕”(Tangled)，且`d1_trend`为“看跌”，`w1_trend`为“宏观下跌趋势”，这与价格站在长期均线之上的事实存在轻微冲突。同时，`adx`为34.84（趋势市），但`ema_slope`为“平坦”，动量描述不一致。 | 中 | 趋势方向判断、交易信心 |
| **“状态”标签不一致**：K线数据中的`bar_status`字段存在大小写不一致的情况（如`Bullish_PINBAR`和`Bearish_Pinbar`），可能影响后续基于规则的信号处理。 | 低 | 数据清洗与自动化处理 |

    - RSI 14 (43.16)：处于“中性偏低”区域，无超买超卖压力，但偏空。
    - MACD：处于零轴下方，确认空头动能占优。
    - ADX (34.84)：显示市场存在一定趋势强度，但非强势趋势。
- **波动率指标**：
    - 布林带：处于“收窄”状态，带宽仅0.95%，预示**波动率压缩，即将选择方向**。
    - ATR (5.44)：绝对值较低，印证市场近期波动较小。
- **高低时间框架冲突**：
    - **短期(M15)**：价格在EMA200附近争夺，布林带收窄，RSI中性，指向**震荡待突破**。
    - **长期(D1/W1)**：被判定为“看跌”和“宏观下跌趋势”，提供**整体偏空背景**。
- **结论**：指标整体描绘出一幅**于长期下跌背景下的低波动盘整**画面。主要矛盾在于长期看跌趋势与价格在关键长期均线（EMA200）上方的盘整之间的拉锯。下一方向性突破（尤其是伴随成交量放大）将至关重要。
