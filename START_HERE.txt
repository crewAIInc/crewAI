QUICK START - 3 STEPS:
----------------------

1. INSTALL OLLAMA
   - Go to: https://ollama.ai/download
   - Download and install Ollama for Windows
   - It will start automatically

2. RUN SETUP
   - Double-click: setup_windows.bat
   - Wait for setup to complete (~5 minutes)

3. INSTALL MODELS
   - Double-click: install_models.bat
   - Choose option 1 (Qwen2.5:32b) or "all"
   - Wait for download (~10-60 minutes depending on choice)

4. TEST IT!
   - Double-click: run_example.bat
   - Enter a topic when prompted
   - Watch your AI agents work!

TROUBLESHOOTING:
----------------

Problem: "Ollama not found"
Solution: Install from https://ollama.ai/download

Problem: "Python not found"
Solution: Install Python 3.10-3.13 from python.org
          Make sure to check "Add Python to PATH"!

Problem: Scripts won't run
Solution: Run Command Prompt as Administrator

Problem: Slow performance
Solution: First run is always slower (loading models)
          Close other GPU applications
          Use smaller models if needed
